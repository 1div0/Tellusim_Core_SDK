AgilitySDK and DXC compiler with shader model 6_8 support is required

The following libraries and headers must be present in the ../d3d12_common/ directory:

* D3D12Core.dll
* dxcompiler.dll
* dxil.dll
* d3d12.h

The latest AgilitySDK and DXCompiler can be downloaded here:

https://www.nuget.org/packages/Microsoft.Direct3D.D3D12
https://www.nuget.org/packages/Microsoft.Direct3D.DXC

.nupkg packages are .zip archives
